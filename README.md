# -Simple-Sales-Dashboard-Design
 Create a basic interactive dashboard that shows sales performance by product, region, and month


##  Tools Used
- **Tableau Public** – for dashboard creation  
- **Microsoft Excel ** – data source  

##  Dataset
**Dataset Name:** Superstore Sales (Sample - EU Superstore)  
**Columns Used:**
- `Order Date`
- `Region`
- `Category`
- `Sales`
- `Profit`

## Dashboard Components
The dashboard consists of **three main visualizations**:

### 1. Sales Over Time (Line Chart)
- Shows total **Sales by Month-Year**
- Helps identify **seasonal trends** and **monthly growth**

### 2. Sales by Region (Bar Chart)
- Compares **total sales and profits** across all regions
- Highlights top-performing regions visually

### 3.  Sales by Category (Donut Chart)
- Displays the **share of sales by Category**
- Created using a **Dual Axis Pie Chart** technique

### 4.  Filter/Slicer
- Added a **Region filter** to interactively view performance by area
